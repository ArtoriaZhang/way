Create Vite + Vue + Typescript ​

Initialize project ​

  • Create project
shell
npm create vite@latest my-vue-app -- --template vue
cd my-project

npm install
npm run dev
  • Install Typescript
shell
npm install typescript
  • We should create a tsconfig.json file in the root folder, like this:
js
{
  "compilerOptions": {
    "target": "esnext",
    "module": "esnext",
    "moduleResolution": "node",
    "importHelpers": true,
    "isolatedModules": true,
    "noEmit": true
  }
}
  • Create a shims-vue.d.tsfile in the src folder, like this:
js
declare module "*.vue" {
  import { defineComponent } from "vue";
  const Component: ReturnType<typeof defineComponent>;
  export default Component;
}
  • Rename the main.js file to main.ts in the src folder, and modify the index.html:
html
 <script type="module" src="/src/main.js"></script>

to

html
 <script type="module" src="/src/main.ts"></script>

Create path alias ​

Should modify both tsconfig.js and vite.config.ts. For example, tsconfig.js:

js
"compilerOptions": {
    "baseUrl": "../",
    "paths": {
      "@/*": ["./*"],
      "@server/*": ["./server/src/*"]
    },
  }

The vite.config.js should be like:

ts
import { defineConfig } from "vite";
import react from "@vitejs/plugin-react-swc";
import path from "path";

export default defineConfig({
  resolve: {
    alias: {
      "@": path.resolve(__dirname, "./"),
      "@server": path.resolve(__dirname, "./server/src"),
    },
  },
  plugins: [react()],
});
